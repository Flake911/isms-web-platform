# SentraISMS

**Centralize. Secure. Comply.**

Your ISO 27001:2022 Command Center — manage risks, controls, audits, and compliance in one platform.

## Quick Start

```bash
# Install & run frontend
cd frontend && npm install && npm run dev
```

Visit http://localhost:3000

## Tech Stack

- **Frontend**: Next.js 16 + React 19 + Tailwind CSS 4
- **Design**: Dark enterprise theme (Stripe/Vercel-inspired)
- **Backend** (planned): NestJS + PostgreSQL + Prisma
